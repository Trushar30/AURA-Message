## ğŸ Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB Cluster
- Groq Cloud API Key

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Trushar30/AURA-Message.git
   cd AURA
   ```

2. **Setup Server**
   ```bash
   cd server
   npm install
   # Create .env file with PORT, MONGODB_URI, JWT_SECRET, GROQ_API_KEY
   npm run dev
   ```

3. **Setup Client**
   ```bash
   cd client
   npm install
   npm run dev
   ```
